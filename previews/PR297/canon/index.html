Decapodes.Canon.Chemistry.GrayScott — Constant

Gray-Scott

Source

A model of reaction-diffusion

Model 

(U, V)::Form0
              
UV2::Form0
              
(U̇, V̇)::Form0
              
(f, k, rᵤ, rᵥ)::Constant
              
UV2 == U .* (V .* V)
              
U̇ == (rᵤ * Δ(U) - UV2) + f * (1 .- U)
              
V̇ == (rᵥ * Δ(V) + UV2) - (f + k) .* V
              
∂ₜ(U) == U̇
              
∂ₜ(V) == V̇
source
Decapodes.Canon.Chemistry.brusselator — Constant

Brusselator

Source

A model of reaction-diffusion for an oscillatory chemical system.

Model

U2V::Form0
              
(U̇, V̇)::Form0
              
α::Constant
              
F::Parameter
              
U2V == (U .* U) .* V
              
U̇ == ((1 + U2V) - 4.4U) + α * Δ(U) + F
              
V̇ == (3.4U - U2V) + α * Δ(V)
              
∂ₜ(U) == U̇
              
∂ₜ(V) == V̇
